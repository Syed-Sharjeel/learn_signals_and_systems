# Learn Signals & Systems

This repository contains **lab tasks, exercises, and supporting MATLAB/Simulink files** for the course **Signals & Systems (EE-232)** offered by the **Department of Electrical Engineering, NED University of Engineering & Technology**.

The repository is created as a **learning aid and reference guide** for juniors and classmates. Content will be **updated weekly** according to the official lab schedule.

---

## Purpose of This Repository

* Provide **organized lab material** for Signals & Systems
* Help students **understand experiments before lab sessions**
* Serve as a **revision and practice resource**
* Share **MATLAB scripts, Simulink models, and reports**

---

## List of Laboratory Experiments

1. Introduction to Oscilloscope, Function Generator, and DC Power Supply
2. Introduction to MATLAB Software
3. (a) Plotting of Basic Signals in MATLAB & Procedural Programming
4. (b) Signal Manipulation and System Development in MATLAB
5. Fourier Series Analysis using MATLAB
6. Charging and Discharging of an RC Circuit
7. RC Circuit Response using MATLAB Simulink
8. Time Domain Analysis of RC Circuit (Zero-Input Response)
9. Time Domain Analysis of RLC Circuit (Zero-Input Response)
10. Convolution Integral of Two Signals (MATLAB vs Analytical)
11. Solving Ordinary Differential Equations (ODEs) using MATLAB
12. Design and Analysis of Low Pass Filter
13. Open-Ended Lab: Design and Analysis of High Pass Filter

---

## Repository Structure

* Each experiment is placed in a **separate folder**
* Folders may include:

  * MATLAB (.m) files
  * Simulink (.slx) models
  * Lab task solutions
  * Supporting notes or screenshots (if required)
